Calculadora de Aluguel de imóvel
================================
# Esta é uma atividade proposta pelo supervisor de aprendizes do BNE [sergio nauffal](https://github.com/sergionauffal)

Nela os aprendizes foram propostos a tarefa de desenvolver uma calculadora para o valor de aluguel de um imóvel.\

As entradas e saidas para o codigo são:\
Entradas:
  -  Area por Metro²
  -  Valor do Metro² da região
  -  Numero de quartos dp imóvel
  -  Se está em um prédio
  -  andar do prédio\
    
Saidas:
  -  Valor de Venda do imóvel
  -  Valor do Aluguel do imóvel
